# NemesaScan V3.4

A fast and lightweight port scanner designed for ethical security testing.  
Supports full port range scanning, progress indicator, and automatic log saving.

## ✦ Features (V3.4)
- Full port scan (1–65535)
- SmartScan mode
- Live progress indicator
- Terminal output + TXT log saving
- Stable and lightweight structure
- No external dependencies
- Works on iSH, Linux, and macOS

## ✦ Usage
Run the scanner:
Enter the target IP or domain when asked.

Results will automatically:
- appear in the terminal  
- save to `NemesaScan_Results.txt`

## ✦ Requirements
- Python 3.x  
No extra modules required.
